## nextCP/2 4.4.6

### Highlights

- **Docker container**: auto-detects the host network interface on first start (no manual bind-interface configuration needed).

### Downloads

- **macOS**: `nextCP2-*.dmg` (installer) or `nextCP2-macos.zip` (portable app image).
- **Windows**: `nextCP2-*.msi` (installer) or `nextCP2-windows.zip` (portable app image).
- **Any OS with a JRE 25**: `nextcp2.jar` (run with `java -jar nextcp2.jar`).
- **Docker**: `docker pull ghcr.io/sf666/nextcp2:4.4.6` (or `:latest`) — see the [Docker install guide](https://sf666.github.io/nextcp2/quick_install/docker/).
- **McIntosh MA9000/MA12000 driver**: `nextcp2-ma9000.jar` (optional).
