## nextCP/2 4.4.1

### Highlights

- **Local player "This device"** — nextCP can now play audio files directly in the browser.
- **Tree search** — search recursively below the current folder.
- **Artist folder support** — display artist covers in search results.
- **Basic AI services** exposed as an MCP server (experimental).
- Improved TCP/IP device driver stability.
- Added Tolgee support for translation services (backend).
- Updated Audio Addict UI support.
- General UI polishing.

### Changes since the last release

- Migrated to Angular 22.
- Switched to JDK 25.
- Moved change detection fully to `OnPush`.
- Increased use of signal-based computing.
- Added an SSE heartbeat.
- Added a waveform bar for the currently playing item.
- Improved error handling while browsing the Content Directory Service (CDS).

### Downloads

- **macOS**: `nextCP2-*.dmg` (installer) or `nextCP2-macos.zip` (portable app image).
- **Windows**: `nextCP2-*.msi` (installer) or `nextCP2-windows.zip` (portable app image).
- **Any OS with a JRE 25**: `nextcp2.jar` (run with `java -jar nextcp2.jar`).
- **McIntosh MA9000/MA12000 driver**: `nextcp2-ma9000.jar` (optional).
