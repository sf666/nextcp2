## nextCP/2 4.4.5

### Highlights

- Platform specific default configuration 
- Native Image Downloads for MacOS, Windows and Docker

### Downloads

- **macOS**: `nextCP2-*.dmg` (installer) or `nextCP2-macos.zip` (portable app image).
- **Windows**: `nextCP2-*.msi` (installer) or `nextCP2-windows.zip` (portable app image).
- **Any OS with a JRE 25**: `nextcp2.jar` (run with `java -jar nextcp2.jar`).
- **Docker**: `docker pull ghcr.io/sf666/nextcp2:4.4.5` (or `:latest`) — see the [Docker install guide](https://sf666.github.io/nextcp2/quick_install/docker/).
- **McIntosh MA9000/MA12000 driver**: `nextcp2-ma9000.jar` (optional).
