## Compatibility
### Browser
This software is tested with the current Chrome browser on desktop and mobile devices, but should also work with
other chromium based browsers like Safari.

### Media Server / Content Delivery Server (CDS)
By now, only the fork of <a href="https://github.com/ik666/UniversalMediaServer">universal media server</a> supports all features and is the recommended media server.
